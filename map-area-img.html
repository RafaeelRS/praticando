Utilizando o Mapeamento de imagens

Para que serve?

A tag map em cojunto com o elemento area, conseguimos criar uma mapa de uma imagem e atribuir dentro da tag area, as coordenadas que correspondem a uma forma que queremos que seja clicável! Portanto, podemos criar uma área com hiperlink dentro de uma img em um espaço específico da imagem.
